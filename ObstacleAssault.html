Obstacle Assault

Status: In Progress · Unreal Engine 5 (Blueprints + C++)

Obstacle Assault is a 3D platformer built as part of GameDev.tv’s Unreal Engine 5 C++ curriculum. I’m using this project to sharpen core gameplay engineering skills: character movement, collision handling, moving/rotating platforms, and polished level flow. While the early implementation is in Blueprints, I’m incrementally translating logic into C++ to mirror production workflows.

What This Demonstrates

  • Gameplay Programming: character control, movement tuning, jump arcs, and fail/respawn logic.
  • Interactive World: spline/rotator-driven platforms, timed obstacles, and checkpointing.
  • UE5 Fundamentals: actors/components, Tick usage, delta time, events, collisions, and traces.
  • Blueprint ⇄ C++: prototyping in Blueprints, then porting systems to C++ for maintainability and performance.

Current Features

  • Moving/rotating platforms with configurable speed/paths.
  • Collision volumes for hazards and reset points.
  • Start/finish flow with restart-on-fall behavior.
  • Work-in-progress: level timing + best-time tracking.

Tech & Tools

  • Engine: Unreal Engine 5
  • Programming: Blueprints, C++ (actors, components, input)
  • Systems: movement, collision, transforms, timers, Tick/delta time
  • Workflow: rapid prototyping in Blueprints → targeted C++ refactors

Note: This project uses course/free assets. I’m sharing screenshots and gameplay footage, but not distributing asset files.
