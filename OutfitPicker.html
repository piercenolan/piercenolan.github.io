Outfit Picker

Status: In Progress

This C++-based outfit picker is designed for individuals with busy schedules who want a streamlined approach to choosing clothes. The goal is to automate wardrobe management while still respecting personal style, recent usage, and (eventually) external factors like weather or profession.

Current Features

  • Users can input clothing items with attributes including:
    • type (e.g., jacket, pants, shoes)
    • isLong (long-sleeved or pants)
    • material
    • color
    • pattern
  • Prompts the user to add or remove clothing items.
  • All data is saved in a .csv file for persistence between sessions.
  • After an outfit is generated, worn items are moved to a separate worn.csv file.
  • Users are asked if laundry has been done to return items to the available pool.

Planned Extensions

  • Integrate real-time weather APIs to recommend season-appropriate outfits.
  • Use user-defined career types (e.g., business, creative, trades) to guide outfit formalness or style.
  • Build a web or mobile interface for more accessible daily use.
  • Enhance outfit selection algorithm based on past usage, aesthetic matching, and constraints.

Technologies Used

  • C++ (OOP, file I/O)
  • CSV-based lightweight database
  • Future: Weather APIs, Web frontend frameworks (e.g., React or Flutter)

This project is actively under development. You can view the source code and follow its progress on GitHub.
