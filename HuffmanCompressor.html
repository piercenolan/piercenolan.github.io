Huffman Compressor (CS106B Final Assignment)

As part of Stanford Universityâ€™s CS106B course on programming abstractions, I implemented a file compression and decompression tool using the standard Huffman coding algorithm. This project required me to build a program capable of encoding and decoding arbitrary files efficiently with lossless compression.

This assignment enabled me to demonstrate proficiency in:

  • Counting character frequencies using a hash map data structure.
  • Constructing a binary tree (encoding tree) via a priority queue, and merging nodes based on ascending frequency.
  • Generating an encoding map from the tree (prefix codes), and producing a compressed data output.
  • Implementing a decoding function to reconstruct input from compressed bitstreams consistently and accurately.

All logic is implemented in C++, using the Stanford C++ library where applicable. The project includes rigorous testing via a provided `huffmanmain.cpp`, ensuring correct functionality across multiple encoding and decoding scenarios. Memory safety and correctness were checked via debugging and manual tests

Note: Source code is restricted per academic policy. Code is available upon request only.
