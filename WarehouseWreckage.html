Warehouse Wreckage

Warehouse Wreckage is a physics-based first-person shooter guided through the GameDev.tv Unreal Engine 5 C++ Developer tutorial series. Using only 20 projectiles, the goal is to inflict maximum damage in an abandoned warehouse environment.

This served as my introduction to Unreal Engine and C++, highlighting my skills in:

  • Spawning actors and managing meshes
  • Implementing physics-based movement with velocity and vectors
  • Integrating simple gameplay logic via Blueprints or equivalent C++ code

All assets used were freely available from Unreal Marketplace, so this page features only screenshots and gameplay footage (no game file is shared).
